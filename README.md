# VELORA – Ecommerce Web Application

## Overview
VELORA is a full-stack ecommerce web application built using Spring Boot, React, and MySQL. It enables users to log in, browse products, and interact with a modern UI powered by real-time backend APIs.

## Tech Stack

### Backend
- Spring Boot
- Spring Data JPA (Hibernate)
- MySQL
- JWT Authentication

### Frontend
- React (Vite)
- Tailwind CSS
- Framer Motion

## Features
- User Login Authentication
- Product Management (CRUD APIs)
- Product Search Functionality
- Image-based Product Display
- Responsive UI Design
- REST API Integration

## Project Structure

Java_FSD_Project/
├── ecommerce-backend/
├── ecommerce-frontend/


## How to Run the Project

### Backend

cd ecommerce-backend
mvn spring-boot:run


### Frontend

cd ecommerce-frontend
npm install
npm run dev
